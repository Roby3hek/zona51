# Описание проекта: Телеграм-бот для приема заявок для пейнтбольного клуба
Данный проект представляет собой телеграм-бота, разработанного на Python с использованием библиотеки pyTelegramBotAPI (telebot). Бот предназначен для автоматизированного сбора заявок от пользователей, желающих организовать активный отдых (гидробол, пейнтбол, лазертаг) в одном из двух городов: Саратов или Энгельс.
